Theme Release: 

	IN LIMBO

Release Version:

	1.0 (nov '04)

Theme Designer: 

	torb / torben rygg

Instructions: 

	1 - Upload the folder named "inLimbo" to your theme directory. 
	2 - Go to your forum, log in as admin and click 
		Administration Panel » Themes 
	    and scroll down to where it says "Import theme." Browse for
	    the file called In+Limbo+by+torb-theme.xmb and upload it.
	3 - Tweakage: If you want to use the original smilies, go to
		Administration Panel » Themes » InLimbo... Theme Details
	    and scroll down to where it says "smilie directory". Change
	    this path to your standard smilie directory (typically this
	    is called /smilies)
	3b- Tweakage: If you want to alter the header file, feel free to
            do so. You will need photoshop in order to alter it. The
	    psd-file is in the root of this compressed archive.

Comments:

	Any comments? Feel free to mail me at torb_@hotmail.com
